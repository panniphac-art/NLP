# ELIZA Chatbot

โปรแกรมแชทบอทง่าย ๆ ที่จำลองการคุยกับนักจิตวิทยา รันบน Terminal ใช้ Python 3 ไม่ต้องติดตั้งอะไรเพิ่ม

## วิธีรัน

```bash
python ELIZA.py
```

## การทำงาน

โปรแกรมรับข้อความจากผู้ใช้แล้วเอาไปจับคู่กับ pattern ที่กำหนดไว้ โดยใช้ Regular Expression ไล่เช็กจากบนลงล่าง ถ้าตรงข้อไหนก็ตอบตามนั้นเลย ถ้าไม่ตรงสักข้อก็จะตอบว่า `PLEASE TELL ME MORE.` แทน

pattern ที่รองรับตอนนี้มีแค่นี้:
- `I'm sad` / `I'm depressed` → แสดงความเห็นใจ
- `I'm ...` อื่น ๆ → ถามกลับว่าทำไมถึงคิดแบบนั้น
- มีคำว่า `all` หรือ `always` → ขอตัวอย่างที่ชัดขึ้น
- `my boyfriend` / `my girlfriend` → ถามว่าแฟนเป็นคนพามาหรือเปล่า

ไม่สนใจตัวพิมพ์เล็ก-ใหญ่ทั้งหมด

พิมพ์ `quit` เพื่อออกจากโปรแกรม

## ข้อจำกัด

รองรับแค่ภาษาอังกฤษ และไม่มีการจำบริบทการสนทนา แต่ละประโยคถูกประมวลผลแยกกันอิสระ
